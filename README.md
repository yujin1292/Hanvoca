# HanVoca
 
 사용자가 원하는 단어들로 구성한 단어장을 생성할 수 있다.
 
 
 ## 암기 기능
 
깜빡이 암기 | 3초 간격으로 단어가 자동으로 움직여 반복학습을 할 수 있다.
---------------------------------------------------------------------------
수동 암기 | 사용자가 직접 단어를 움직여 단어를 암기할 수 있다.
